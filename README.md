# Symbolic Control Framework

A modular, layered framework for N-dimensional symbolic controller synthesis with LLM-based specification generation and hardware-accelerated parallelization.

## Features

- **N-Dimensional Support**: Works with any state dimension
- **Arbitrary Dynamics**: Support for any robot model via pluggable dynamics
- **LLM Integration**: Convert natural language to formal automata
- **Hardware Acceleration**: Multi-core CPU and GPU parallelization
- **Modular Architecture**: Clear separation of concerns for easy extension
- **Interactive Web Interface**: Draw workspace, run simulations, visualize results
